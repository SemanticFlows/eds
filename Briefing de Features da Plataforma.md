
**1. Core de Apostas (placar como eixo central)**

- Interface principal orientada a **eventos ao vivo**
- Listagem de **ligas/jogos com odds dinâmicas**
- Apostas baseadas em **fatores de placar** (tempo, resultado parcial, eventos)
- Atualização contínua (quase em tempo real)

**2. Acompanhamento em Tempo Real**

- Painel de **placar ao vivo com timeline**
- Segmentação por períodos (ex: minutos do jogo)
- Visualização contínua para suporte à decisão de aposta

**3. Expansão de Apostas**

- Mesma lógica de placar aplicada a:
    - **Pré-jogo**
    - **Mercados específicos (props, variações de odds)**
- Interface reutiliza o modelo base para diferentes tipos de aposta

**4. Cassino e Jogos Integrados**

- Sessão de **cassino ao vivo**
- Jogos instantâneos (slots, roleta, etc.)
- Integração com sistema de saldo unificado

**5. Sistema de Missões**

- Missões categorizadas por áreas da plataforma
- Objetivos com **recompensas (créditos, bônus)**
- Progressão orientada a engajamento do usuário

**6. Loja / Economia Interna**

- Loja com itens/benefícios (ex: boosts, vantagens)
- Uso de moeda interna ou créditos promocionais
- Incentivo à recorrência

**7. Sistema de Progressão (Gamificação)**

- Rank (ex: Bronze → níveis superiores)
- Barra de progresso e milestones
- Benefícios atrelados ao nível

**8. Gacha / Recompensas Diárias**

- Sistema de **roleta (gacha)** com prêmios
- Incentivo de login diário
- Distribuição de bônus aleatórios

**9. Torneios**

- Competições com ranking
- Prêmios por performance
- Engajamento competitivo entre usuários

**10. Central de Informações**

- Página com:
    - Níveis e benefícios
    - FAQ
    - Explicação de mecânicas
- Suporte à compreensão do sistema

**11. Sistema de Promoções**

- Campanhas com:
    - Cashback (ex: 50%)
    - Créditos promocionais
    - Eventos especiais
- Destaque visual na interface

**12. Arquitetura de UX**

- UI modular e consistente
- Foco em:
    - **ação rápida (apostar)**
    - **feedback visual imediato**
- Camadas:
    - Interface (UI)
    - Lógica de apostas (core)
    - Sistemas de engajamento (gamificação + promoções)

**Resumo estrutural**  
A plataforma combina:

- **Motor de apostas em tempo real (placar-driven)**
- **Camada de entretenimento (cassino + gacha)**
- **Camada de retenção (missões, ranks, promoções)**
